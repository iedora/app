# iedora-frontend

iedora frontend — Bun-workspaces monorepo; UI-only Next.js 16 app serving menu.iedora.com (multi-tenant menu builder + auth) and iedora.com (landing) over a host-based rewrite. All data, auth and business rules live in iedora-backend (Go). Built and deployed by Coolify.
